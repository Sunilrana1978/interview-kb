REST API Headers

// cheat_sheet.http — v1.0

Common Request Headers
Header Purpose Example
Authorization Auth token sent to the server to verify identity Bearer xyz123
Content-Type Media type of the request body application/json
Accept Expected response format from the server application/json
User-Agent Identifies the client application making the request PostmanRuntime/7.0
Cache-Control Directives for caching mechanisms no-cache
Cookie Send stored cookies to the server sessionId=abc
Host Domain name of the target server api.example.com
Accept-Language Preferred language(s) for the response en-US
Common Response Headers
Header Purpose Example
Content-Type Media type of the response body application/json
Content-Length Size of the response body in bytes 348
Set-Cookie Instructs client to store a cookie token=abc; HttpOnly
Cache-Control Cache policy for the response max-age=3600
Access-Control-Allow-Origin Specifies allowed CORS origins *
Authorization Auth token or credentials returned by server Bearer <token>
ETag Unique identifier for cache validation "abc123"
Server Software info about the origin server nginx
