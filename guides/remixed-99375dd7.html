08
MONITORING, ERROR HANDLING & REPLAY
📡
Consumer Lag Monitoring
Alert if consumer group lags >1hr. Track offset progress per group. Detect stuck consumers.
🔁
Replay from Kafka
Reset consumer offset to replay events. 7-day retention window. No data loss on failures.
☠️
Dead Letter Queue
Failed transforms go to DLQ topic. Ops reviews daily. Fix rule, replay from original topic.
🌙
Nightly Reconciliation
2AM batch job compares PLM vs ERP vs WMS state. Flags drift. Publishes correction events.
🔒
Idempotency
Event ID deduplication. Same event processed twice = no duplicates. Safe retries.
📋
Schema Registry
All event schemas versioned. v1.0 and v1.1 supported simultaneously during migration.
